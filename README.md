# API de Gestão de Contratos - Climbe Investimentos

Esta é a API para o sistema de gestão de contratos da **Climbe Investimentos**. Ela foi desenvolvida para gerenciar e fornecer acesso aos dados relacionados aos contratos da empresa.

---

## Requisitos

Para rodar o projeto localmente, você precisa ter o **Node.js** instalado em sua máquina.

---

## Como Rodar

Siga os passos abaixo para iniciar o servidor da API:

1.  **Instale as dependências:**
    Abra o terminal na pasta raiz do projeto e execute o seguinte comando:
    ```sh
    npm install
    ```

2.  **Inicie o servidor:**
    Após a instalação das dependências, rode o comando abaixo no mesmo terminal:
    ```sh
    node index.js
    ```

A API estará rodando e pronta para ser usada.
